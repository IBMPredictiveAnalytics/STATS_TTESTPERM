STATS TTESTPERM Extension Command

Introduction

This extension procedure provides permutation t tests of equality of means for one variable and two groups or two variables and one group. The test does not assume a normal distribution for the data, and it can do an unequal variance (Welch) test, which is the default, or an equal-variance test. For general information on permutation tests, see the reference below.

This procedure does not support case weights. They will be ignored with a warning. Cases with missing data will be excluded listwise. The tested variables must have a scale (continuous) measurement level, and the group definition variable must have a nominal or ordinal level. The dialog box appears under Analyze > Compare Means and Proportions as Two Group or Two Variable Permutation t Test.

The output includes the standard, i.e., nonpermutation, t test information that will match the standard t test procedure in the two-group case. Since the two-group case is also covered in the Regression > Permutation Tests procedure, STATS PERM, when the variance assumption is set to equal, those results should be close, but since permutation tests have some randomness due to the sampling, the reaults are unlikel to be exactly equal. In fact, if you run this procedure multiple times, the output will vary slightly from run to run.

This procedure supports split files. Any splits that are too small or, for the two-group test, where only one group appears, are automatically excluded with a warning message. If there are many splits, the Organize Output by Groups setting, SPLIT FILES SEPARATE, will produce more readable output.

Syntax

STATS TTESTPERM
PAIRED = NO** or YES
VARIABLE1 = First Test Variable*
VARIABLE2 = Second Test Variable
GROUPVAR = group variable
GROUPVALS = group definition values
ALTERNATIVE = TWOSIDED** or GREATER or LESS
CONFLEVEL = confidence level
EQUALVAR = NO** or YES
QQPLOT = YES** or NO
NPERM = number of permutations

/HELP
STATS TTESTPERM /HELP displays this information and does nothing else.

* Required
** Default


STATS TTESTPERM VARIABLE1=salary GROUPVAR=gender
GROUPVALS = "m" "f" QQPLOT=NO.

Details

PAIRED specifies whether to do a two variable (paired) test or a two group test.

VARIABLE1 specifies the first test variable

VARIABLE2 specifies the second test variable for a two variable test. It is required if PAIRED is YES.

GROUPVAR specifies the group definition variable for a two group test. It is required if PAIRED is NO.

GROUPVALS specifies the two group definition values for a two group test. It is required if PAIRED is NO. Values should be quoted if GROUPVAR is a string variable.

EQUALVAR specifies whether to assume equal variance for the two variable or group distributions.

NPERM specifies the number of permutations. The default is 9999. Larger numbers mean greater accuracy but more execution time.

ALTERNATIVE specifies whether the alternative hypothesis is two sided, the group 1 mean is greater than group 2 or the group 1 mean is less than group 2. For the two variable case, "greater" means that the first variable mean is greater than the second variable. For the two group case, the group with the smaller mean is considered to be the first.

CONFLEVEL specifies the test confidence level and defaults to 0.95.

QQPLOT specifies whether to do a Q-Q plot of the two distributions or not. The Q-Q plot is helpful beyond the t test in determining whether the two distributions are the same. If they are identical, the points would form a straight line on the diagonal.
