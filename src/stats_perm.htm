Introduction

This extension procedure provides permutation t tests of equality of means for one variable and two groups or two variables and one group. The test does not assume a normal distribution for the data, and it can do an unequal variance (Welch) test, which is the default, or an equal-variance test. For general information on permutation tests, see the reference below.

This procedure does not support case weights. They will be ignored with a warning. Cases with missing data will be excluded listwise. The tested variables must have a scale (continuous) measurement level, and the group definition variable must have a nominal or ordinal level. The dialog box generates syntax for the STATS PERMTTEST extension command.

The output includes the standard, i.e., nonpermutation, t test information that will match the standard t test procedure in the two-group case. Since the two-group case is also covered in the Regression > Permutation Tests procedure when the variance assumption is set to equal, those results should be close, but since permutation tests have some randomness due to the sampling, the reaults are unlikel to be exactly equal. In fact, if you run this procedure multiple times, the output will vary slightly from run to run.

Main Tab

Permutation Test Type Choose whether to do a two-group or two variable test.

First Test Variable For either type, select a scale variable.

Second Test Variable For a two-variable test, select another scale variable.

Grouping Variable For a two group test, specify a categorical group variable.

Group1 and Group2 Specify the two values that define the groups. If the variable is a string, enclose the values in quotes.

Options Tab

Variance Assumption Specify whether to do an unequal variance (Welch) test or an equal variance test.

Alternative Hypothesis Choose whether to a two-sided, one greater than two, or one less than two test. For the two variable case, "greater" means that the first variable mean is greater than the second variable. For the two group case, the group with the smaller mean is considered to be the first.

Number of Permutations Choose the number of permutations. Larger numbers mean greater accuracy but more execution time.

Q-Q Plot Check this box to include a Q-Q plot of the two variables or two groups. This is helpful beyond the t test in determining whether the two distributions are the same. If they are identical, the points would form a straight line on the diagonal.

Split Files

This procedure supports split files. Any splits that are too small or, for the two-group test, where only one group appears, are automatically excluded with a warning message. If there are many splits, the Organize Output by Groups setting in the Split Files dialog will produce more readable output.
